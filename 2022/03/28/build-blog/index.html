用hexo在github上搭建个人blog

发表于 2022-03-28 更新于 2022-03-29 分类于 博客搭建

注册github账号

  1. 点击右上角+号，选择New repository
  2. Owner 默认为你的账号 / Repository name 那一栏填上looksth.github.io（looksth替换为你自己的name）
  3. 选择Public
  4. 选择Add a README file
    最后确定

    下载必要软件及文件

  5. 下载git
    • 官方地址：https://git-scm.com/
    • 安装对应的版本
    • 输入：
      git -v
      显示对应的版本则说明安装成功
  6. 下载node.js
    • 官方地址：https://nodejs.org/en/download/
    • 安装对应的安装包
    • 安装好后，在git终端内 输入：
      node -v
      显示对应版本则说明安装成功
  7. 安装hexo
    • 在git终端上输入命令：
      npm install -g hexo-cli
    • 安装完后输入
      hexo -v
      查看版本

      git配置ssh key

      配置ssh key是为了方便后续上传到github时不需要每次输入密码啥的
      一般 .ssh 文件是在C:\Users\looksth\ 下 looksth替换为你自己的用户
      输入命令
      ssh-keygen -t rsa -C "github绑定的邮箱地址"。
      一路回车即可
      进入到.ssh文件夹打开id_rsa.pub复制里面的内容
      接着打开github主页，点击个人设置，点击左侧的SSH and GPG keys，点击New SSH key
      将id_rsa.pub复制的内容粘贴到key中，title随便起一个就行
      配置账号和密码
